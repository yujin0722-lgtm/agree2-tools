# AGREE II 診療ガイドライン評価ツール

診療ガイドラインをAGREE II(Appraisal of Guidelines for Research and Evaluation II)の日本語訳(Minds, 2022年9月改訂)に基づいて評価するための、Webベースの評価・集計ツールです。Googleフォームを使わず、Google Apps Script(GAS)とGoogle スプレッドシートをバックエンドに、GitHub Pagesでホストした静的HTMLページから利用します。

勉強会などのグループワークで、複数の班(2〜4班)がそれぞれAGREE IIの23項目+全体評価に回答し、その結果をダッシュボードでレーダーチャートやコメント一覧として確認できます。

## 公開ページ

GitHub Pagesを有効にすると、以下のURLで利用できます(`(ユーザー名)`と`(リポジトリ名)`は実際の値に置き換えてください)。

| ページ | URL | 共有範囲 |
|---|---|---|
| ハブページ | `https://(ユーザー名).github.io/(リポジトリ名)/` | 誰でも(参加者・関係者) |
| 評価フォーム | `https://(ユーザー名).github.io/(リポジトリ名)/hyoka.html` | 参加者 |
| ダッシュボード | `https://(ユーザー名).github.io/(リポジトリ名)/dashboard.html` | 参加者・関係者 |
| 管理画面 | `https://(ユーザー名).github.io/(リポジトリ名)/kanri.html` | **主催者のみ(共有しない)** |

## ファイル構成

```
├── index.html          # ハブページ(評価フォーム・ダッシュボードへの入口、使い方の説明)
├── hyoka.html           # 評価フォーム(参加者が班ごとに回答)
├── dashboard.html        # 結果ダッシュボード(レーダーチャート、コメント一覧、PDF出力)
├── kanri.html            # 管理画面(ガイドライン名・班数の設定、新しい勉強会の開始など)
├── Code.gs               # バックエンド(Google Apps Scriptにコピーして使用)
└── セットアップ手順.md     # 初回セットアップの手順書(非エンジニア向け)
```

`Code.gs`はこのリポジトリには実行環境がなく、Google スプレッドシートに紐づくApps Scriptプロジェクトに貼り付けて使います。詳細は`セットアップ手順.md`を参照してください。

## 初めて使う方へ

セットアップは最初の1回だけ必要です。手順は `セットアップ手順.md` にまとめてあります。プログラミングの知識がなくても進められる内容です。

セットアップが終われば、日常の運用(ガイドライン名・班数の設定、勉強会終了後のリフレッシュ)はすべて管理画面(`kanri.html`)から行えます。スプレッドシートを直接開く必要はありません。

## 使い方の概要

- **参加者**: `hyoka.html` を開き、班を選んで23項目+全体評価に回答します。「一時保存」で途中保存、「提出(確定)」で確定します。確定後の修正は「確定を取り消す」を押してから行います。
- **関係者・参加者**: `dashboard.html` で、班ごとの領域別得点(レーダーチャート)、コメント、未回答項目の有無を確認できます。PDFとして保存も可能です。
- **主催者**: `kanri.html` で、ガイドライン名・班数の設定、班数変更時のデータ整理、勉強会終了後の「履歴に保存して新しい勉強会を開始する」操作を行います。

## 技術構成

- **フロントエンド**: 素のHTML/CSS/JavaScript(ビルド不要、GitHub Pagesでそのまま配信)
- **バックエンド**: Google Apps Script によるWebアプリ(`doGet`/`doPost`)
- **データストア**: Google スプレッドシート(設定・回答・評価履歴の3シート)
- **グラフ描画**: Chart.js(レーダーチャート)
- **PDF出力**: html2canvas + jsPDF

## 注意事項

- `kanri.html` のURLは参加者に共有しないでください。アクセス制限(パスワードなど)は設けていないため、URLを知っている人は誰でも設定変更ができます。
- 過去の勉強会の詳細な回答は、スプレッドシートの「評価履歴」シートに保存されます。
